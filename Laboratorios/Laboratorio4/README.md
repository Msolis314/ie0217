# README Laboratorio 4
En el presente laboratorio se exploró la complejidad de tiempo de los algoritmos: 
- BubleSort
- SelectionSort
- InsertionSort
- QuickSort

El archivo algoritmos.cpp contiene la definición de los algoritmos y el archivo measureTime.cpp contiene lo correspondiente al cálculo del tiempo.

## Compilación 

Para compilar ejecutar el siguiente comando:
```bash
g++ -o sorting.exe .\main.cpp .\measureTime.cpp .\algoritmos.cpp
```
La ejecución se realiza mediante:
```
./algoritmos.exe
```
